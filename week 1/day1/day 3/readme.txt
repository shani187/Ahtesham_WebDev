# Positivus Landing Page Clone

Ahtesham Ali


## What I Learned

During this project, I learned:

- How to convert a Figma design into HTML structure
- How to use **Flexbox** for horizontal alignment
- How to use **CSS Grid** for card layouts
- How to manage spacing using padding and margin
- How to style buttons, cards, and sections professionally
- How to organize code for readability and maintenance
- How to work with images and icons in web design
- How to match colors, typography, and dimensions from a reference design

## Challenges Faced

Some challenges I faced while building this project:

- Matching the exact spacing and proportions of the Figma design
- Adjusting image sizes to fit correctly in sections
- Making cards look identical to the original design
- Aligning navbar items properly
- Recreating icons and illustrations without original assets
- Maintaining consistent styling across all sections
- Achieving a near pixel-perfect layout
